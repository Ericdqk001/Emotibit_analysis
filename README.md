# Setting up

## Install uv

* On Windows
`powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"`

* On macOS
`curl -LsSf https://astral.sh/uv/install.sh | sh`

## Install 🤖 Just a command runner

* On Windows
`winget install --id Casey.Just --exact`

* On macOS
`brew install just`

## Run (in a new CLI)

`just setup`
